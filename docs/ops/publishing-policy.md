# Publishing Policy

> コンテンツ公開のルール、チャネル別戦略、透明性原則を定義する。

---

## Channel Strategy

### Substack (Primary — English)

- **Weekly Flagship**: Weekly Audit（毎週月曜/火曜公開）
  - 週間予測精度レビュー
  - モデルパフォーマンス分析
  - 注目トレンド・インサイト
- **Daily Signals**: STRONG ピックのみ厳選公開
  - 全予測を垂れ流さない。高信頼度のものだけを配信
  - 読者の信頼を守るため、量より質を徹底
- **Special Articles**: 開幕予測、プレイオフ展望、モデル解説など

### note.com (Japanese — 独自価値)

- **EN記事の日次ミラーではない**
- **JA独自価値を提供する**:
  - 週次フォーカス記事（Weekly Auditの翻訳ではなく、日本読者向けに再構成）
  - 日本人選手の分析（大谷、ダルビッシュ、吉田など）
  - セイバーメトリクス解説（日本語での教育コンテンツ）
  - MLB観戦ガイド（日本のファン向け）
- **頻度**: 週1-2本。質を維持できる範囲で

### X (@MoneyballDojo)

- **Morning (試合前)**: 本日の厳選シグナル（STRONGピックのみ）
- **Midday**: 試合中間アップデート、注目の展開
- **Evening**: 結果レキャップ、的中/外れの振り返り
- **スレッド形式**: 深い分析はスレッドで展開

## Transparency Principle

### 全予測は試合開始前にログする

**これは最も重要なルール。**

- Google Sheets に予測を記録するタイミングは、必ず試合開始前
- 後出しジャンケンは絶対にしない
- 公開予測と内部記録が一致することを保証する
- 読者が検証可能な形でデータを提供する

### 実装

```
run_daily.py (08:00 UTC) → Google Sheets 書込（タイムスタンプ付き）
                         → 試合開始は通常 17:00-23:00 UTC
                         → 予測は常に試合の数時間前に確定
```

## Publish Modes

予測の公開には3つのモードがある。Publish Triage（Layer 2）が判定する。

### `publish` (Default)

- **条件**: 通常の運用日。モデルに問題なし
- **動作**: Morning Brief → 自動的に公開準備 → X投稿
- **CEO操作**: 不要（Slack通知で確認のみ）

### `caution`

- **条件**: モデル精度に懸念がある場合
  - 直近の的中率が閾値を下回っている
  - データソースに異常がある（例: Spring Training と Regular Season の境界）
  - 新しい特徴量やモデル変更の直後
- **動作**: CEOにSlack通知 → CEO判断を待つ
- **CEO操作**: `publish` に変更するか `hold` に変更するか判断
- **デフォルト**: CEOが24時間以内に応答しない場合 → `hold` として扱う

### `hold`

- **条件**: 重大な問題がある場合
  - モデルが明らかに壊れている（エラー率が異常に高い）
  - データソースが完全にダウン
  - セキュリティ問題
- **動作**: 公開を完全停止 → CEOに緊急Slack通知
- **CEO操作**: 問題解決を確認してから `publish` に変更
- **原則**: hold の解除はCEOのみが行える

## Content Quality Standards

### 予測コンテンツ

- ティア（STRONG / MODERATE / LEAN）を必ず明記
- 予測の根拠（主要な特徴量）を簡潔に説明
- 過去の精度を定期的に開示
- 「これは予測であり、ベッティングのアドバイスではない」免責を含める

### 分析コンテンツ

- データに基づく主張のみ行う
- 主観的意見はそれと明記する
- ソースデータ（MLB Stats API）を参照可能にする

### 日本語コンテンツ (note.com)

- 英語コンテンツの直訳ではなく、日本読者向けに再構成
- MLBに馴染みのない読者にも分かる説明を心がける
- 日本人選手の文脈を積極的に取り入れる

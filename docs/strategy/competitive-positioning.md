# Competitive Positioning — Moneyball Dojo

## Landscape

### 1. AI Picks Accounts on X

- 高勝率主張（"80%+ win rate"など）
- トラックレコード不透明、事後選択バイアスが横行
- 外れた予測は削除、当たった予測だけスクショで宣伝
- 有料Discord/Telegramへの誘導が目的

### 2. 既存セイバーメトリクスサイト

- **FanGraphs**: 豊富なデータ、projection systems（ZiPS, Steamer）、記事
- **Baseball Savant**: Statcastデータの宝庫、リーグ最高のビジュアライゼーション
- **Baseball Reference**: 歴史的データの決定版
- **共通点**: データは豊富だが、日々の予測配信はしていない

### 3. 有料ピックサービス

- **Action Network**: プロベッターのpicks、有料壁の向こう
- **Covers, Dimers, etc.**: アルゴリズムベースだがプロセス非公開
- **共通点**: 検証困難、透明性が低い

---

## Moneyball Dojoの立ち位置

```
            データ深度
              ↑
  FanGraphs   |   Baseball Savant
  Baseball Ref|
              |
              |        ★ Moneyball Dojo
              |        (データ + 予測配信 + 透明性)
              |
  AI picks    |   Action Network
  accounts    |   Covers, Dimers
              +-------------------------→ 予測配信
```

- Picks servicesとdata sitesの**間**に位置する
- 無料で予測を公開 → 信頼構築 → 有料は"意思決定の圧縮"に価値を置く
- 日本語展開で競合ゼロのニッチを獲得

---

## Competitive Advantages

### 1. 透明性（最大の差別化要因）

- 事前ログ: 全予測をfirst pitch前にGoogle Sheetsに記録
- 事後監査: Weekly Auditで全結果を公開レビュー
- 全履歴公開: 誰でもいつでもトラックレコードを検証可能
- 「都合の悪い結果を隠さない」ことが信頼の源泉

### 2. バイリンガル（EN + JA）

- 英語圏: 競合は多いが、透明性で差別化
- 日本語圏: MLB予測をデータドリブンで配信するサービスがほぼ存在しない
- 大谷翔平効果で日本のMLB関心は過去最高
- 二言語展開で市場規模を実質2倍に

### 3. テクニカル（モデルの中身を見せる）

- モデルアーキテクチャ、特徴量、学習データを記事で説明
- セイバーメトリクス + ML の交差点を教育コンテンツとして提供
- "ブラックボックス"ではなく"ガラスボックス"

### 4. 継続改善（改善プロセス自体がコンテンツ）

- 週次監査で改善点を公開
- モデルの失敗と修正を記事にする
- 読者がモデルの進化を一緒に見守る体験

## What We Don't Compete On

| 領域 | 理由 |
|------|------|
| **速報性** | ESPN, Rotowireには勝てない。速さは我々の武器ではない |
| **データ量** | FanGraphs, Savantには勝てない。生データの網羅性では戦わない |
| **ベッティング攻略** | "これで勝てる"は我々のブランドではない。リサーチデスクであり、tipping serviceではない |
| **エンタメ性** | Jomboy, Foolish Baseballのようなエンタメ路線ではない |

---

## 競合が真似しにくいこと

1. **事前ログの蓄積**: 始めた日からの全記録。後からは作れない
2. **二言語コンテンツの継続**: EN+JAを毎週出し続けるのは参入障壁
3. **技術的透明性**: モデルの中身を見せる勇気。多くのサービスはブラックボックスを好む
4. **失敗の公開**: 外れた予測の分析をコンテンツにする。これをやりたがるサービスは少ない
